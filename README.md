# G-Retro - API

This project consists of creating a tool for teams to hold retrospective meetings.

## Commands

Typescript:

```sh
npm run tsc
```

Commitizen:

```sh
npm run commit
```

Eslint:

```sh
npx eslint FILEPATH
```

Run API(ts-node-dev):

```sh
npm run dev
```

Run GraphQL Code Generator:

```sh
npm run codegen
```

Docker setup:

```sh
docker compose build
```

Docker run:

```sh
docker compose up -d
```
